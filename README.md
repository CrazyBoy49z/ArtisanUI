CLI User Interface for Laravel Artisan.

## Installation

You can install the package via composer:

```bash
composer require vladreshet/artisanui
php artisan vendor:publish --provider=VladReshet\\ArtisanUI\\ArtisanUIServiceProvider
```

## Usage

``` bash
php artisanui
```
or
``` bash
./artisanui
```
